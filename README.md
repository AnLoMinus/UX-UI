# UIUX-CraftVault

**Crafting Interfaces. Shaping Experiences.**

UIUX-CraftVault הוא מאגר־על לשלדי UI/UX ותבניות ממשק מוכנות לשימוש, הבנויות ב־HTML/CSS/JavaScript טהור. זהו Vault אחד מרכזי שבתוכו פועלים אוספים (Collections) — כל אוסף הוא עולם עיצובי עצמאי עם אופי, חוקים וכיוון משלו, אך כולם נשענים על ליבה משותפת כדי לשמור על סדר, סקייל ותחזוקה חכמה לאורך זמן.

---

## חזון

* לבנות ספרייה מודולרית ועקבית של תבניות UI/UX שמאפשרת לשכפל, להתאים ולפרוס במהירות.
* לשמור על סטנדרט עיצובי אחיד תוך מתן חופש יצירתי לכל אוסף.
* לאפשר תצוגה חיה, תיעוד ברור ותרבות עבודה בריאה לקהילה.

---

## שימוש מהיר

1. שיבטו את המאגר.
2. פתחו את `index.html` בדפדפן כדי לראות את אתר הבית והמסמכים המרכזיים.
3. עברו לתיקיות האוספים (כשתתווספנה) או לתיעוד הקצר להתחלה מהירה.

```bash
git clone <repo-url>
cd UIUX-CraftVault
open index.html
```

---

## תיעוד מקוצר

* **הסכמות שמיות:** כל אוסף כולל `_identity.md`, תיקיית `_starter`, ותבניות תחת `templates/`.
* **מבנה תבנית בסיסי:**

```
index.html
styles.css
app.js
meta.json
README.md
preview.png
```

* **ניהול גרסאות:** SemVer, מתועד ב־`CHANGELOG.md` ובקובץ `VERSION`.
