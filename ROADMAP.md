# מפת דרכים

המסמך מתאר כיוונים עתידיים ומיילסטונס עיקריים. התאריכים משוערים ויתעדכנו עם ההתקדמות.

## מיילסטון 1 — קטלוג MVP (רבעון 1, 2026)
- הקמת ליבה (`core/`) לטוקנים, ריסט בסיסי ותבניות UI בסיסיות.
- יצירת 3 אוספים ראשונים עם `_identity.md` ותבניות ראשוניות.
- אתר תצוגה עם פילטרים לפי אוסף ותגיות.

## מיילסטון 2 — הרחבת תבניות (רבעון 2, 2026)
- הרחבת ספריית תבניות (דשבורד, לוגין, פרופיל, מסחר).
- שיפור שדות `meta.json` והוספת בדיקות עקביות.
- מערכת צילומי מסך אוטומטית לתבניות.

## מיילסטון 3 — קהילה ואוטומציה (רבעון 3, 2026)
- תהליך Contributor ברור עם CI בסיסי.
- חבילות Starter ל־Figma/Sketch.
- תיעוד מתקדם למעבר בין אוספים.

## מיילסטון 4 — ערכת Design System (רבעון 4, 2026)
- ייצוא tokens ל־CSS/JSON.
- ספריית Components עם Storybook.
- CLI ליצירת אוספים ותבניות.
